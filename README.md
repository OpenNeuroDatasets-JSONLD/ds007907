README
======

Dataset title
-------------
Elevated translocator protein in skull bone marrow is linked to human chronic pain


Description
-----------
This dataset contains multimodal neuroimaging data acquired from patients with chronic low back pain,
patients with knee osteoarthritis, and healthy controls. Data include positron emission tomography
(PET) with the radiotracer [11C]PBR28 and structural MRI (T1-weighted).


Participants
------------
Participants included patients with chronic low back pain, patients with knee osteoarthritis, and
healthy control subjects.


Imaging modalities
------------------
- PET: [11C]PBR28, a radiotracer that binds the 18 kDa translocator protein
  (TSPO), a marker of neuroinflammation/immune cell activation. Acquired as
  a 90-minute dynamic scan on a Siemens Biograph mMR PET/MR scanner; static
  SUV images were reconstructed from the 60-90 minute post-injection window.

- Structural MRI: T1-weighted MPRAGE, acquired on the same Siemens Biograph
  mMR scanner at 3T, coregistered to the PET image.


De-identification
-----------------
All structural MRI images (T1w) were defaced prior to public release to remove facial features and
protect participant privacy.


Study design
------------
This is an observational, cross-sectional study. No experimental intervention or task was performed
during image acquisition.
